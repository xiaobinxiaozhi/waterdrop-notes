# 水滴笔记 · PWA 版

护眼温润的笔记应用，支持手机和电脑通用。

## 使用方法

### 方式一：部署到 JVS 云文件空间

1. 将本文件夹所有文件打包或逐个上传到你的云文件空间
2. 在手机或电脑浏览器中打开 `index.html` 的预览链接
3. 添加到主屏幕：浏览器菜单 →「添加到主屏幕」，就像原生 App 一样用

### 方式二：本地直接使用

直接双击 `index.html` 在浏览器中打开即可（仅限本机，不支持云同步）

## 功能特点

- 📝 富文本编辑（加粗/斜体/标题/列表/引用）
- 🏷️ 标签管理
- 💧 子文档系统（选中文字转子文档，悬停预览）
- 🌙 护眼主题（亮色/暗色）
- ☁️ 云同步（跨设备笔记同步，利用文件空间）
- 📱 PWA 支持（可安装到手机主屏幕，离线可用）

## 云同步使用说明

1. 点击工具栏的 ☁️ 云同步 按钮
2. 选择同步方式：
   - **手动同步**：导出 JSON 文件，上传到文件空间，其他设备导入
   - **自动同步**：配置数据文件 URL，一键拉取合并
3. 配置好后，手机和电脑可共用同一份笔记数据
